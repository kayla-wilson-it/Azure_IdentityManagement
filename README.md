# Azure Identity Management

This repository contains all screenshots and documentation for **Azure Active Directory / Microsoft 365** hands-on tasks. These tasks demonstrate managing users, groups, role assignments, and Multi-Factor Authentication (MFA) as part of Azure Identity Management practice.  

## Skills Demonstrated
- Managing Azure AD users & groups  
- Assigning roles (e.g., **User Administrator**, **Security Reader**)  
- Implementing Conditional Access policies  
- Enabling Multi-Factor Authentication (MFA)

## Hands-On Tasks
1. Created users and groups in Azure Active Directory  
2. Assigned roles to users (User Administrator, Security Reader)  
3. Enabled MFA for selected users
